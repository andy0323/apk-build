# apk-build 

apk-build node 一款安卓打包apk的工具 

[![npm version](https://badge.fury.io/js/apk-build.svg)](http://badge.fury.io/js/apk-build)

## Test

如何测试ipa-build？

打开终端，输入`npm test`，等打包完毕，输入`open .`，就可以看到生成完毕的signed.apk
